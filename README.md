# Instagram Large Feed

GitHub Pages에서 바로 실행되는 정적 웹 앱입니다.

## Pages 배포

1. GitHub 저장소의 `Settings > Pages`로 이동합니다.
2. `Build and deployment`에서 `Deploy from a branch`를 선택합니다.
3. `Branch`는 `main`, 폴더는 `/ (root)`로 설정합니다.

## 구성

- `index.html`: 3x1, 3x2, 3x3 선택 화면
- `row-1.html`: 3x1 생성
- `row-2.html`: 3x2 생성
- `row-3.html`: 3x3 생성

브라우저에서 이미지를 합성하므로 서버가 필요 없습니다.
